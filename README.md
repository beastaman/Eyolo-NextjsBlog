
# Eyolo - React Nextjs Personal Blog Template


Eyolo! A modern and feature-rich personal blog template built with React, NextJS, and Tailwind CSS. With Eyolo, you can build a professional blog with minimal effort. Itâ€™s support MDX and Markdown; you can use any react component within your blog-post. Eyolo is SEO friendly and it has built-in RSS feed support. It has working AJAX contact form which is directly send mail to your email.


## Features

- 100 score on PageSpeed Insights and GTmetrix
- 3 Homepage versions
- Easily Setup Headless CMS
- Built-in support for Markdown and MDX
- Custom Instant Search
- Support author in post
- SEO Friendly
- Working AJAX contact form
- RSS Feed
- Supports Sitemap
- Newsletter (Formspree)
- Fully mobile responsive
- Table of contents
- Code highlighting
- Youtube & Vimeo video
- Image-popup with caption
- Image gallery
- Text quotations
- Social Share
- Copy link to clipboard
- Helpful documentation

## Run Locally

Clone the project

```bash
  git clone https://github.com/beastaman/Eyolo-NextjsBlog.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
